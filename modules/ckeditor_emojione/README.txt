INTRODUCTION
------------
This module Integrates the CKEditor Emojione plugin to CKEditor for Drupal 8.

Allows users to add various emojis using a CKEditor dialog.

INSTALLATION
------------
1. Download the plugin from https://ckeditor.com/addon/emojione.
2. Place the plugin in the root libraries folder (/libraries), so that
   plugin.js will be located at DRUPAL_ROOT/libraries/emojione/plugin.js.
3. Enable CKEditor Emojione in the Drupal admin.
4. Configure your WYSIWYG toolbar to include the button.

REQUIREMENTS
------------
CKEditor Module (Core)
